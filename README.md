## 🔧 Local Preview

1. **Clone this repo**

   ```bash
   git clone https://github.com/pals6/pals6.github.io.git
   cd pals6.github.io
   ```

2. **Serve locally**

   You can use any static-server. For example with Python:

   ```bash
   # for Python 3.x
   python -m http.server 8000
   ```

3. **Open**

   Navigate to `http://localhost:8000` in your browser to see the site.
